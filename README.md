### Running the project

Clone this repository :

```console

git clone https://github.com/jrlherm/Marvel-frontend--LeReacteur.git
cd Marvel-frontend--LeReacteur
```

Install packages :

```console
yarn install
```

When installation is complete, run the project :

```console
yarn dev
```
